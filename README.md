# Desafio Técnico TutorMundi

## Desafio: Criar uma LLM e uma pesquisa
#### Contexto: A pesquisa consisti em que passos e perguntas eu faria para melhorar a LLM e adequa-la a task do TutorMundi
#### A LLM será para analisar uma pergunta no aplicativo do TutorMundi e responder de qual matéria pertence aquela pergunta

## O que foi desenvolvido:
#### A pesquisa, ou melhor dizendo seu planejamento e minhas análises, está no link a seguir: [Notion](https://www.notion.so/Pesquisa-TutorMundi-fcfdbe9af015406e953c47e65bff1312?pvs=4)
#### Já o treinamento da LLM está aqui neste repositório
- Link para baixar o melhor modelo treinado: [Drive](https://drive.google.com/drive/folders/1YsWEgU6M3xZFlWIkZ2kpkBYrYVc4n3Eb?usp=sharing)
- As perguntas originais juntamente com as extras estão na pasta Dados
- Na pasta Notebooks você achará o notebook usado para treinamento
  - Juntamente com um .py de execução, ou ao final do notebook de treinamento
- A pasta de modelos está vazia por conta do tamanho do modelo gerado
  - No entanto ao baixar o modelo treinado do drive, ou gerar o modelo por si mesmo, é na pasta "Model/Modelo/" que ele deve ser armazenado
